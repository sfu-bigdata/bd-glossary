Data Literacy: Glossary and Index

Glossary

ADVANCED RESEARCH COMPUTING

Advanced research computing (ARC) provides massive computational horsepower and storage in a cloud environment to handle problems and data that are too complex for a single desktop computer. 1,2 #disciplines

ALGORITHM

A sequence of instructions telling a computer how to answer a specific question. 3,4 #processing

ANALYTICS

The process of using statistical models and software to transform data into useful information and to draw conclusions towards effective decision making.5,6 #disciplines

ARTIFICIAL INTELLIGENCE

A branch of computer science that allows machines to acquire and apply knowledge to handle new inputs and analyze patterns to solve diverse problems. 7 #disciplines

BEHAVIOUR(AL) ANALYTICS

A type of business analytics that examines behavioural data about people to understand how and why individuals act the way they do and to make more accurate predictions for future behaviour.8,9 #data-analysis

BIG DATA

Refers to the massive amounts of data generated around the world that is too large, complex or varied for traditional processing software. Its potential to be analyzed for valuable information is enabled by technology such as advanced research computing. 10,11 #disciplines

CAUSAL INFERENCE

Determines whether observations made in one variable are the reason for an effect observed in another variable, possibly occurring at a later time. See also correlation mining and predictive analytics. #data-analysis

CLASSIFICATION

An approach in machine learning where a program is trained with labeled data to determine which category a new observation belongs to. See also clustering. #data-analysis

CLOUD COMPUTING

Access to data, storage, applications, and other computing resources made available to many users on-demand over the Internet to improve scalable computing power and reliability. 12,13 #disciplines

CLUSTER

A network of computers (or compute nodes) that work on tasks together in parallel. The concept is different from clustering.

CLUSTERING

A machine learning technique that groups similar data points together to uncover hidden structure. Unlike classification, clustering does not require labelled training data and is a type of unsupervised learning. #data-analysis

COMPUTING WITH DATA

Learning from data using computing tools and programming languages, such as Python or R with large ecosystems of libraries providing a convenient level of abstraction and statistical methods that can be organized into data processing pipelines.14 #disciplines

CORRELATION MINING

As a measure of association between two variables, correlation can be used for prediction and indicate the presence of causal relationship. It does, however, not sufficiently imply causation. 15 #data-analysis

CYBERSECURITY

A model in information security designed to govern and evaluate how an organization handles data when it is stored, transmitted or processed. This model emphasizes that data should not be accessed without authorization, should not be altered or compromised without authorization, and should be accessible upon legitimate request. See Secure Computing. 16 #disciplines

DATA

Collection of examples, observations, measurements, facts, points, or other items of information that can be represented in structured or unstructured form.

DATA CLEANING

Careful removal of erroneous or unreliable data points. #processing

DATA EXPLORATION AND PREPARATION

Exploratory data analysis (EDA) is a formative step in the creation of models. Views of the data are used to learn about patterns or relationships among variables. This includes data cleaning and manipulation for further analysis. #processing

DATA INTEGRATION

The process of combining information from different data sources in preparation for data processing. 17 #processing

DATA MINING

An analytical process where large datasets are explored or “mined” in search of meaningful patterns, relationships or insights. The process can include statistics, machine learning or other forms of artificial intelligence.18,19 #disciplines

DATA REPRESENTATION

The form in which data are stored, processed and transmitted such that its information content and context are retained as much as possible. Choices of form are influenced by hardware, software or other constraints around processing and analysis resources. 20 #processing

DATA SCIENCE

A multidisciplinary activity combining programming skills, math and statistical analysis, and sector-specific expertise to extract insights from data. Often performed in stages: 1. Data Exploration and Preparation, 2. Data Representation and Transformation, 3. Computing with Data, 4. Machine Learning, 5. Data Visualization and Presentation. 21,22 #disciplines

DATA TRANSFORMATION

The process of converting data from one form into another to gain better insight. For instance, converting unstructured data into structured form that can be analyzed further. #processing

DATABASE

An organized collection of data that allows easy access, management, updating and analysis of data. Commonly used databases are MySQL, PostgreSQL, as well as various NoSQL options.23 #storage

DESCRIPTIVE ANALYTICS

An initial stage of data processing that involves creating a summary of historical data with the goal of answering the question, “What happened?”24,25 #data-analysis

DISTRIBUTED FILE SYSTEM

A mechanism that stores files on servers and allows clients, with permission, to store and process files as if they were stored on their own computer.26,27 #storage

FEATURE

A key property that characterizes a data point (representing some real-world object) in the context of a machine learning problem or other type of analysis.

FEATURE ENGINEERING

A human-driven process of finding the most important features to develop predictive models. Formerly, as “art” to the “science” of machine learning, big data increasingly moves towards automated representation learning.

FLEXIBLE SCHEMA

Unlike SQL databases with a tabular structure, objects or documents stored in a flexible schema can be different from one another.28 #data-types

GENERATIVE MODELING

Fitting a model that can generate synthetic data beyond given observations. For example, a language model trained on a collection of text can be used to suggest possible words to modify or continue a given phrase. 29

GOODNESS OF FIT

Summarizes how well the values observed in the data agree with those values expected by the model. #processing

INFORMATION

Meaning encoded in data that answers questions to better understand a concept by interpreting data within the context of its problem setting or domain. See also data analysis algorithms and information visualization. 30

INSIGHT

Actionable information gained by interpreting data analysis results, helping people to make more informed decisions. 31

IOT (INTERNET OF THINGS)

A system of connected, “smart” objects, including smartphones, wearables and smart-appliances, that collect and exchange information without requiring human interaction. The IoT is a huge generator of data.32,33 #storage

KNOWLEDGE DISCOVERY

Aims to extract insight from data in databases. It involves a number of steps including the evaluation and possibly interpretation of patterns to gather insight and knowledge.34 #disciplines

MACHINE LEARNING

A part of artificial intelligence that enables machines to learn from experience to perform certain tasks by using algorithmic models that are trained to imitate patterns present in data to support prediction, data generation and other forms of data analytics. #disciplines

METADATA

Also called “data about data,” it provides a structured description and context for a data point - document, image, or file - to help organize, find and understand the data. 35 #data-types

MODEL FITTING

Optimizes how well a machine learning model can make predictions for previously unseen test data.

NETWORK

A system where multiple computing devices are connected to each other to exchange information and resources through a data link. For example, the Internet.36,37

NETWORK ANALYSIS

Mapping and measuring the relationships between people, groups, organizations, computers and other connected entities. It is used to simplify complex relationships, to make them easier to analyze. 38

NO-SQL DATABASE

A type of database designed to handle large volumes of data that may not have a structure. #storage

PREDICTIVE ANALYTICS

A process for analyzing current data to determine future events or other unknowns. Related to descriptive and prescriptive analytics, it draws on techniques from data mining, modeling, machine learning and statistics.39,40

PRESCRIPTIVE ANALYTICS

A type of analytics with the goal of using data to determine the best course of action for a specific scenario. 41 #data-analysis

REGRESSION

A type of statistical model that predicts numerical values (instead of class labels). #data-analysis

RELATIONSHIP MINING

Relationship mining examines associations between two or more variables in a dataset, for example, by correlation mining and causal mining. #data-analysis

SECURE COMPUTING

Efforts to ensure privacy and to protect data, devices and computing systems from harm such as hacking, damage, and malpractice, and mitigate service disruptions. 42,43 #disciplines

SQL

SQL is a language that works with structured, tabular data allowing to query and construct such data. It forms a standard for many database systems. [28] #processing

STRUCTURED DATA

Data that is organized into clearly defined fields associated with variables or attributes, such as dates, words, or numbers that are recorded for each observation or item. Items are often represented as rows of tabular spreadsheets that can be stored in a database for easy processing and analysis.44,45 #data-types

TABULAR DATA

Data items or observations that are organized as rows that contain values under columns that correspond to specific variables or properties. 46,47 #data-types

TEXT MINING

A process of analyzing text to capture key concepts, themes, relationships and trends.48 #data-analysis

UNSTRUCTURED DATA

Data that is not organized in a pre-defined way under a single data model. Examples include text, images, audio, or video. 49,50 #data-types, #lesson-01

VISUALIZATION

Visual representation of data including plots, charts, maps and infographics to support people with tasks such as sensemaking, effective communication of information, and improved pattern detection. For example, flight operation using a radar screen that displays sensor data in real-time. 51,52 #data-analysis

VISUALIZATION DASHBOARD

Multiple interactive graphical views of data that would be too complex if presented in a single visualization, for instance, to support complex narratives and insights by showing key performance indicators (KPIs) for monitoring of a business or other processes. #data-analysis

Categories

The categories offer an alternative grouping that helps to find related terms.
